# 01. 应用架构与生命周期

## 1. 概述

### 1.1 领域定位

本文档覆盖 VCP Mobile 前端层的**应用级骨架**——从浏览器（WebView）加载到用户可见界面的完整链路。该领域不负责具体业务功能（如聊天、Agent 设置），而是承载以下横切关注点：

- **应用入口编排**：Vue 实例创建、Pinia / Router / 指令注册、样式系统挂载
- **启动引导状态机**：权限检查 → 后端核心就绪等待 → 业务数据预加载 → READY
- **全局 DOM 层级秩序**：背景层、主内容、抽屉、覆盖层、启动屏的 Z-Index 语义化堆叠
- **物理按键拦截**：Android 返回键的三层消费模型与 Modal History Stack（Operation Aegis）
- **生命周期桥接**：WebView 的 isibilitychange 与 Android pause/resume/stop 事件的跨层映射

该领域是前端所有业务模块的**物理底座**。任何覆盖层、抽屉、页面栈的层级冲突，或启动时序的竞态条件，都应首先回归此处诊断。
